Курс по WEB-разработке

Курс подразумевает, что студенты уже освоили базовый python.

В рамках курса даются азы html/css достаточные для формирования html страниц браузеру, но не достаточные для профессиональной верстки.

Глубина освоения django будет достаточной для самостоятельного написания web-проектов, которые содержат редактируемые страницы, структуру специфических данных, отображение их на фронтенде и создание-редактирование этих данных.

Все разделы django будут освещены.

Программа курса

Занятие Тема Содержание
1 Основы html/css. GIT
  • html - теги и свойства
  • css - свойства и селекторы
  • Bootstrap

  • введение и описание фреймворка, старт проекта
